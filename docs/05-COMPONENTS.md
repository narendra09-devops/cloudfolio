# Future UI Components

This document defines responsibilities for future CloudFolio interface components. It does not define implementation code.

## Navigation

### Navbar

Provides primary navigation, responsive menu behavior, active page state, recruiter hub access, and theme controls.

### Footer

Provides secondary navigation, contact links, social links, copyright, license reference, and repository link.

### Breadcrumbs

Shows page hierarchy on nested project, blog, architecture, certification, and case study pages.

## Page Sections

### Hero

Introduces Narendra's positioning, target roles, primary skills, and high-value actions such as viewing the recruiter hub or resume.

### Section Header

Provides consistent titles, descriptions, and optional actions for major content sections.

### Call to Action

Guides visitors toward resume, recruiter hub, contact, or featured technical evidence.

## Content Cards

### Project Card

Summarizes a project with title, problem area, stack, outcomes, links, and status.

### Case Study Card

Highlights business or operational problem, engineering approach, measurable result, and reading time.

### Blog Card

Summarizes technical posts with topic, date, category, tags, and excerpt.

### Certification Card

Displays certification name, issuer, date, verification status, and related skills.

### Architecture Card

Previews an architecture diagram with domain, systems involved, and complexity level.

## Resume Components

### Experience Timeline

Shows career progression, roles, responsibilities, achievements, and technologies.

### Skill Matrix

Organizes skills by domain such as cloud, containers, IaC, CI/CD, monitoring, programming, Linux, and networking.

### Achievement List

Highlights measurable accomplishments and senior-level outcomes.

### Resume Download

Provides access to current resume variants when supported.

## Technical Evidence Components

### Architecture Diagram Viewer

Displays diagrams with summaries, legends, related content, and accessibility descriptions.

### Metrics Highlight

Shows quantified outcomes such as uptime, cost savings, deployment frequency, recovery time, or automation impact.

### Technology Tag

Labels content by tools, platforms, and domains.

### GitHub Widget

Displays repository, contribution, or project metadata from GitHub integrations when added.

## Interaction Components

### Button

Supports clear actions with consistent priority levels and accessible labels.

### Link

Handles internal, external, document, and contact navigation consistently.

### Filter Controls

Allows users to filter projects, posts, certifications, and diagrams by topic, stack, or role relevance.

### Search

Supports future content discovery across resume, projects, posts, case studies, and architecture.

### Contact Form

Collects recruiter, collaboration, or general messages with validation, privacy guidance, and secure handling.

## Recruiter Components

### Recruiter Summary Panel

Shows seniority, target roles, countries, availability signals, core skills, and resume action.

### Role Fit Matrix

Maps Narendra's experience to target roles such as SRE, Platform Engineer, DevOps Engineer, Cloud Infrastructure Engineer, and AWS Cloud Engineer.

### Interview Topics

Summarizes strong discussion areas for hiring teams.
